# Магазин тематических рюкзаков и сумок для школы

## О проекте
Проект создан для лабораторной работы по Django. Реализован простой интернет-магазин с тремя страницами.

## Страницы
1. **Главная** — содержит ссылки на другие страницы.
2. **Об авторе** — информация о студенте.
3. **О магазине** — описание тематики (рюкзаки и сумки для школы).
